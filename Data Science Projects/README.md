## 🎯 Projects Overview

### 1. Pakistan Population Analysis
**Objective**: Analyze demographic trends in Pakistan focusing on population growth and gender distribution.

**Key Features**:
- Annual Growth Rate (AGR) analysis across provinces
- Gender distribution analysis (urban vs rural)
- Statistical summaries and comparative visualizations
- Province-wise demographic insights

**Technologies**: Python, Pandas, Matplotlib, NumPy

**Output**: Statistical tables, multi-panel visualizations, key demographic insights

**Best For**: Understanding population trends, growth patterns, demographic distribution

---

### 2. Titanic Survival Analysis
**Objective**: Analyze historical Titanic disaster data to understand survival factors.

**Key Features**:
- Gender-based survival analysis
- Passenger class impact evaluation
- Interaction effects (gender × class)
- Historical context and findings
- Visualization of survival patterns

**Technologies**: Python, Pandas, Matplotlib, NumPy, Seaborn

**Output**: Statistical tables, multi-panel visualizations, historical insights

**Best For**: Learning about classification analysis, understanding the impact of categorical variables, historical data analysis

---

### 3. Twitter Sentiment Analysis
**Objective**: Analyze public sentiment across topics using Twitter data.

**Key Features**:
- Sentiment classification (Positive, Negative, Neutral)
- Topic-based sentiment breakdown
- Engagement pattern analysis
- Cross-tabulation analysis
- Comprehensive sentiment visualizations

**Technologies**: Python, Pandas, Matplotlib, NumPy, Seaborn

**Output**: Sentiment statistics, topic-sentiment relationships, engagement metrics, visualizations

**Best For**: Understanding sentiment analysis, topic modeling, social media analysis, opinion mining

---

## 🚀 Quick Start

### Prerequisites
- Python 3.7+
- pip (Python package manager)

### Installation

1. **Clone or download this repository**
   ```bash
   cd Data\ Science\ Projects
   ```

2. **Install required packages**
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

### Running Individual Projects

#### Pakistan Population Analysis
```bash
cd "Pakistan Population Analysis"
python annual_growth_rate_analysis.py
python gender_distribution_analysis.py
```

#### Titanic Survival Analysis
```bash
cd "Titanic Survival Analysis"
python titanic_analysis.py
```

#### Twitter Sentiment Analysis
```bash
cd "Twitter Sentiment Analysis"
python sentiment_analysis.py
```

## 🐛 Troubleshooting

### Common Issues

**Issue**: "Module not found" error
```bash
pip install pandas numpy matplotlib seaborn
```

**Issue**: CSV file not found
- Ensure data files are in the correct directory
- Check file names match exactly (case-sensitive)
- Verify relative paths in scripts

**Issue**: Visualizations not displaying
- Try `plt.show()` at the end of scripts
- Check backend settings if using in notebooks
- Use `%matplotlib inline` in Jupyter notebooks
